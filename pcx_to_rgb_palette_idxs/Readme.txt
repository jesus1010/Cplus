Build Info
----------
  1. Create 'build' folder, at some CMakelist.txt directory level.
  2. 'cd build'
  3. 'cmake -DCMAKE_BUILD_TYPE=debug ../' or 'cmake -DCMAKE_BUILD_TYPE=release ../'

    [+] debug build, enable logger and debug symbols
    [+] release build, enable O2 optimizations and disable logger

  4. Target directory store build
